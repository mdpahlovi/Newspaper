# Newspaper

A modern application for managing articles.

## Run Locally

Clone the project

```bash
  git clone https://github.com/mdpahlovi/Newspaper
```

Go to the project directory

```bash
  cd Newspaper
```

Install dependencies

```bash
  npm install
```

Run with npm

```bash
  npm run dev
```
